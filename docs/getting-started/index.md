# Getting Started

Squirrel Backend is the Python service behind the Squirrel snapshot/restore tooling for EPICS control systems. This section walks you from a fresh clone to a running stack.

Pick the page that matches what you need:

- **[Installation](installation.md)** — full setup guide. Docker Compose (recommended) and local development workflows, troubleshooting, and a Docker command reference.
- **[Configuration](configuration.md)** — `SQUIRREL_*` environment variables and the `docker/.env` Docker Compose configuration.
- **[API Keys](api-keys.md)** — bootstrap your first key, manage keys via REST or scripts.

Once the stack is up:

- **[API Reference](../api-reference/index.md)** — REST and WebSocket endpoints
- **[Architecture](../architecture/index.md)** — services, processes, and data model
