# eed-swapps-template
Template repository for projects made by the Software Applications team in EED.

## Setup Steps

1. Find references of `eed-swapps-template` and replace them with the project name
2. Add required packages to `requirements.txt` and `environment.yml`
3. Update `.github/workflows/run-tests.yml` with special testing configurations
    - For example, PyQT requires a special configuration for testing on Mac OS
